Privacy

Privacy Policy

Last updated: 14 July 2026

The short version

PinFolder collects nothing, stores nothing outside your Mac, and sends nothing anywhere. There are no accounts, no analytics, no telemetry, no crash reporting, and no network requests of any kind in the app.

What the app stores — on your Mac only

  • Your pinned paths live in a plain-text file at ~/.pinned-folders, one path per line. It never leaves your machine.
  • "Pin on Top" creates ordinary symlinks next to your files. "Pin to Sidebar" uses the macOS sidebar-favourites system.
  • The "start at login" preference is stored by macOS itself (SMAppService).

Deleting these files, or running pinfolder-setup uninstall / make uninstall, removes everything the app ever created except your own pins file and shortcuts, which stay yours.

What we can see

Nothing. PinFolder has no server, and the developer receives no information about you or your files — ever.

This website

  • The site is hosted on GitHub Pages. GitHub may log basic visit data (such as IP addresses) as described in GitHub's privacy statement.
  • Fonts are loaded from Google Fonts, which means your browser requests font files from Google. See Google's privacy policy.
  • The site itself sets no cookies and runs no analytics.

Installing via Homebrew

If you install with brew install sai-na/tap/pinfolder, Homebrew downloads the source from GitHub and may collect its own anonymous usage analytics — that's Homebrew's behaviour, not PinFolder's. You can disable it with brew analytics off.

Changes

If this policy ever changes, the update will be published on this page with a new date. Given what PinFolder is — a local, open-source utility — don't expect much to change.

Contact

Questions? Open an issue on GitHub or reach the developer via the links on the home page.
